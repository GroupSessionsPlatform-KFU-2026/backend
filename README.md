# Group Sessions Platform — Backend

## Description
Backend service built with FastAPI for virtual collaborative study rooms.

The system allows:
- Creating study sessions
- Real-time whiteboard (WebSocket)
- Chat between participants
- Pomodoro timer

## Roles
- Moderator (creates and manages session)
- Participant (joins session and interacts)

## Tech Stack
- FastAPI
- WebSocket
- uv (package manager)
- ruff (linter & formatter)
